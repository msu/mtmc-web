MTMC-16 Documentation

MonTana Mini-Computer - 16-bit Educational System

This 16-bit educational computer system implements the x366 assembly language, providing a complete environment for learning low-level programming concepts.

Getting Started

  • Quick Start Guide beginner

    Start here if you're new to assembly programming or the MTMC-16 system. Learn the basics of how computers work, understand registers and memory, and step through your first programs in the emulator.

  • X366 Programmer's Guide intermediate

    Comprehensive tutorial that teaches x366 assembly through progressive examples. Covers output, loops, functions, arrays, strings, recursion, and memory allocation.

Technical Reference

  • X366 Architecture Specification reference

    Complete technical specification for the x366 instruction set. Covers registers, memory layout, addressing modes, instruction encodings, and system calls.

  • Binary Format Specification advanced

    Documentation of the x366 executable binary format including headers, section layout, and encoding details. For assembler developers.

  • Function Calling Convention intermediate

    Guide to x366 calling conventions: parameter passing (6 registers), stack frames, prologue/epilogue, and return values.

Specialized Topics

  • Graphics Programming intermediate

    Visual programs using the 160x144 pixel display with 4-color palette. Drawing primitives, manual refresh, animation patterns, and frame rate control.
